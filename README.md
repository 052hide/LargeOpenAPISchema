# LargeOpenAPISchema

## Directories

### _build

バンドル後のOASファイル

### api

バンドル前のOASファイル
スキーマ修正対象

### client

TypeScriptコード生成要configファイルと生成後のファイル

## Scripts

### OASバンドル

```sh
yarn bundle:oas
```

### TypeScriptコード生成

```sh
yarn generate:client
```

### OAS schemaとexamplesの整合性チェック

```sh
yarn generate:client
```
